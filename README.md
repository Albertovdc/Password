# Loops
## For Loop
````python
fruits = ["peach", "apple", "pear"]
for fruit in fruits:
    print(fruit)
fruit += # choice from another list
# Output
# peach
# apple
# pear
````
## Range function
````python
for number in range(1, 100, 2):
    print(number)
# but not the last digit
# The third digit make the step
````
